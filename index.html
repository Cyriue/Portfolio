Pokédex Interactif

L'application Pokemon est une application web développée avec Angular. Elle permet aux utilisateurs de consulter une liste de Pokemon, d'afficher les détails de chaque Pokemon et d'effectuer une recherche par nom.
L'application est composée de plusieurs fonctionnalités clés. Tout d'abord, la page d'accueil affiche une liste de Pokemon, avec des informations de base telles que leur nom, leur image et leurs types. Les utilisateurs peuvent cliquer sur un Pokemon pour accéder à une page de détails spécifique à ce Pokemon.
La page de détails affiche des informations plus détaillées sur le Pokemon sélectionné, telles que ses points de vie (HP), ses points de combat (CP) et sa date de création. Les utilisateurs peuvent également revenir à la liste principale ou effectuer une recherche par nom à partir de cette page.
L'application utilise des services et des composants Angular pour gérer la logique et l'affichage des données. Les données des Pokemon sont récupérées à partir d'une source de données, telles qu'une API ou un fichier JSON, et sont ensuite affichées dans les composants appropriés.

Carte Interactive

Le projet "Sport" est une application web interactive développée avec JavaScript, jQuery et d'autres technologies.
Il permet d'explorer les installations sportives de la ville de Toulouse via une carte interactive basée sur la bibliothèque Leaflet. Les données sont récupérées de manière efficace à partir de l'API de Toulouse Métropole grâce à un fichier de configuration.
Ce projet combine des compétences front-end et back-end pour offrir une interface professionnelle, incluant la manipulation de données JSON et l'organisation de l'architecture MVC pour assurer une structure modulaire et maintenable.

Opus musiques

La refonte du site OPUS Musiques est un projet ambitieux visant à moderniser et améliorer l'expérience utilisateur du webzine musical toulousain dédié à la scène locale.
À l'aide des technologies telles que Tailwind CSS, Laravel, HTML, CSS et JavaScript, plusieurs aspects clés ont été repensés et développés.
Au niveau du front-end, les vues ont été retravaillées pour offrir un design attrayant et une navigation intuitive aux utilisateurs. Les améliorations apportées permettent de mettre en valeur les actualités, les clips, les sorties de CD et les concerts, offrant ainsi une expérience riche et immersive.
Le panel administrateur a également été remanié, offrant une interface conviviale et fonctionnelle pour la gestion du contenu du site. Les fonctionnalités de publication, de modification et de suppression ont été optimisées, simplifiant ainsi le travail de l'équipe éditoriale.
La base de données a été repensée selon un modèle conceptuel de données (MCD) adapté aux besoins spécifiques du site OPUS Musiques. Cette structure permet une gestion efficace et organisée des informations, favorisant la rapidité et la précision des recherches et des requêtes.
Dans l'ensemble, cette refonte du site OPUS Musiques utilise des technologies modernes pour offrir une expérience utilisateur améliorée et une plateforme web performante dédiée à la découverte de la scène musicale locale de Toulouse.
